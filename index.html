    Set is a visualization skill and focus game. (help?)

    built by @fkadev
    • A set is three cards with following rules:
    • All numbers must be same or must be different in a set.
    • All colors of numbers must be same or must be different in a set.
    • All shapes must be same or must be different in a set.
    • All colors of shapes must be same or must be different in a set.
